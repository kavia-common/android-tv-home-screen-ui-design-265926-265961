# android-tv-home-screen-ui-design-265926-265961

Android TV Home screen UI prototype
- Executive Gray theme
- Top fixed bar
- 3+ horizontally scrollable rows of large movie cards
- D-pad navigation with visible focus ring and scaling
- Bottom info/preview pane updates on focus
- Static mock data; no network calls

How to run
- Open the `android_tv_home_screen_frontend/` project in Android Studio
- Select a TV emulator (Android TV)
- Build and Run

Notes
- Buttons are non-functional, provide visual feedback only.
- Poster images use colored placeholders for performance and offline demo.
